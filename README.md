# Tip-Calculator-Project
The Tip Calculator is a simple Python console application that calculates how much each person should pay when splitting a bill, including a selected tip percentage. The program takes user input for the total bill amount, desired tip percentage, and the number of people sharing the bill. It then computes the total bill with the tip included and divides it evenly among the group.

This project demonstrates fundamental Python programming concepts such as user input handling, data type conversion (int and float), arithmetic operations, and rounding values for accurate financial calculations.

Key Features:
	•	Accepts bill amount from the user
	•	Allows selection of tip percentage
	•	Splits the bill among multiple people
	•	Calculates and displays the final amount each person should pay
	•	Rounds the result to two decimal places for clarity

This project is part of the 100 Days of Code – Python Bootcamp, designed to strengthen understanding of Python basics and build confidence in creating interactive command-line applications.
